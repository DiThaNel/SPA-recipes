## Recipe Card Planner

A dynamic recipe management application designed to streamline meal planning. Users can create detailed recipe cards and organize them across a weekly schedule using intuitive drag-and-drop functionality.

### Core Features
- Recipe Management: Create, edit, and view recipes with dedicated fields for Name, Description, Preparation steps, and Ingredients.

- Interactive Scheduler: A drag-and-drop interface to organize recipes by days of the week.

- Smart Export: Generate and export a consolidated list of ingredients for easy grocery shopping.

### Technical Stack
- Frontend: React & Next.js

- Styling: Bootstrap

- Animations: Framer Motion (for smooth drag-and-drop transitions)

- Testing: Jest

### How to run
- npm install
- npm run dev 
- http://localhost:3000
